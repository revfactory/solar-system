## 1. 프로젝트 개요

### 목표
태양계 8개 행성의 궤도 운동과 외관을 사실적으로 시뮬레이션하는 인터랙티브 3D 웹사이트를 제작한다.

### 기술 요구사항 요약
- 실시간 3D 렌더링 (WebGL/WebGPU)
- 과학적으로 정확한 궤도 계산 (JPL 데이터 기반)
- 사실적 행성 텍스처, 대기, 고리 렌더링
- 시간 제어 (재생/가속/되감기/날짜 이동)
- 행성 정보 표시 및 인터랙션
- 반응형 (데스크톱 + 모바일)

### 3.4 구현 알고리즘 핵심

**케플러 방정식 풀이 (Newton-Raphson):**
```javascript
function solveKepler(M, e, tol = 1e-12) {
  let E = M + e * Math.sin(M);
  for (let i = 0; i < 50; i++) {
    const dE = (E - e * Math.sin(E) - M) / (1 - e * Math.cos(E));
    E -= dE;
    if (Math.abs(dE) < tol) return E;
  }
  return E;
}
```

**궤도요소 → 3D 좌표 변환**, **시간 변환(UTC→JD→T)**, **외행성 보정** 등 전체 구현 가이드는 `_workspace/02_orbital_mechanics_data.md` 섹션 8 참조.

### 5.2 기능 우선순위 매트릭스

**필수:**
- 3D 행성 렌더링 + 텍스처
- 궤도 라인 시각화
- 태양 렌더링 (발광)
- 시간 제어 UI
- 행성 클릭 → 정보 패널
- 카메라 자유 이동 + 행성 간 포커스 이동
- 실시간 천체 위치 반영
- 반응형/모바일

**권장:**
- 대기 셰이더, 토성 고리, 위성 표시
- 별 배경, 행성 자전, 소행성대
- 블룸/글로우, 태양 그림자

**선택:**
- 카이퍼벨트, 우주 탐사선 궤적, 별자리, VR/AR

### 5.3 UX 패턴 (Best Practices)

| 패턴 | 설명 | 레퍼런스 |
|------|------|---------|
| 천체 목록 사이드바 | 좌측에 행성 목록, 클릭 시 자동 카메라 이동 | NASA Eyes |
| 정보 슬라이드인 패널 | 우측/하단에서 슬라이드, 탭 구분 (개요/물리/탐사) | Solar System Scope |
| 하단 타임라인 바 | 재생/일시정지 + 속도 조절 + 날짜 표시 | NASA Eyes |
| 초기 뷰 = 태양계 조감도 | 첫 로딩 시 전체 조감도 | 전체 공통 |
| 더블클릭 = 포커스 | 천체 클릭 시 카메라 자동 이동+줌 | 전체 공통 |

### 5.4 차별화 포인트

4. **모바일 퍼스트** — 터치 UX 최적화
