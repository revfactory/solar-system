# Paper Analysis — 행성과학 논문 분석 스킬

행성과학 분야의 학술 논문을 체계적으로 검색, 분석하여 연구 동향과 핵심 발견을 정리한다.

## 논문 검색 전략

### 검색 소스 (우선순위)
1. **NASA ADS** (ads.harvard.edu) — 천문학/행성과학 최대 DB
2. **arXiv** (arxiv.org/list/astro-ph.EP) — 행성과학 프리프린트
3. **Google Scholar** — 범용 학술 검색
4. **AGU/EGU 학회** — Geophysical Research Letters, JGR Planets

### 검색 전략
1. 행성명 + 핵심 키워드로 WebSearch (예: "Mars subsurface water recent discovery")
2. NASA ADS에서 최근 5년 고인용 논문 검색
3. 주요 논문의 참고문헌에서 추가 핵심 논문 추적 (snowball search)
4. 리뷰 논문(review article)을 먼저 찾아 분야 전체 지형도를 파악

## 논문 분석 프레임워크

각 논문에 대해 다음 구조로 분석한다:

```markdown
### [논문 제목] (저자, 연도)
- **저널:**
- **DOI/링크:**
- **연구 배경:** 이 연구가 해결하려는 문제
- **핵심 방법론:** 사용 데이터(탐사선, 망원경 등)와 분석 기법
- **주요 발견:** 3~5개 핵심 결과
- **과학적 의의:** 기존 이해를 어떻게 바꾸었는가
- **한계점과 향후 과제:**
- **관련 행성:** [해당 행성명]
```

## 연구 동향 분석

논문 분석 후 행성별로 다음을 종합한다:

1. **연구 타임라인:** 주요 발견의 시간순 정리
2. **패러다임 변화:** 기존 이론이 뒤집힌 사례
3. **현재 논쟁:** 학계에서 합의에 이르지 못한 주제
4. **미래 전망:** 계획된 미션이 답할 수 있는 질문

## 품질 기준
- 행성당 최소 5편의 핵심 논문을 분석한다
- 최근 10년 논문을 우선하되, 패러다임 전환 논문은 연도 무관
- 피어리뷰 논문을 우선, 프리프린트는 "(프리프린트)"로 명시
- 논문 원문 접근 불가 시 초록과 인용 정보 기반으로 분석하되, 이를 명시한다
